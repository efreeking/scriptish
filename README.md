# Scriptish

[![Gitter chat](https://badges.gitter.im/scriptish/scriptish.png)](https://gitter.im/scriptish/scriptish)

## 关于

Scriptish 是一个 Firefox 的用户脚本管理器，从 Greasemonkey 分支出来的。

## 支持的浏览器

* Firefox 21+
* Seamonkey 2.8+
